# PoB Goods tracking

![pobgoods view](/fl-darkbot/index_assets/pobgoods.png)

First you need to add PoBs for tracking! [See how to do it here](./tracking_bases.md). Then you can add pob goods for tracking :) Or alerts.

PoB good tracking makes possible to track goods only on matched bases added via `base` group of commands

**IMPORTANT! Pob good you add for view or alerts, should be exposed to forum/darkstat view. [see here](https://darkstat.dd84ai.com/#how_to_turn_pob_feature_on) for more details.**

- `. pobgood tags list` - check tracked pob goods
- `. pobgood tags add commodity_food` - for all matched bases track `commodity_food` onboard
- `. pobgood tags remove commodity_food` - remove `commodity_food` tag
- `. pobgood tags clear` - remove all pobgood tags from tracking
